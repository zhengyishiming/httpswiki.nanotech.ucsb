Amscope Quickstart Usage Guide

From UCSB Nanofab Wiki
Jump to navigation Jump to search

A number of our microscopes now have 14MP Amscope cameras with accompanying free Amscope software installed.

The software has already been calibrated for measurements on each microscope, users should not perform any calibrations without consulting Staff first.

Start Imaging Session

  1. Open the "AmScope" software, via Desktop icon or Start-Bar icon:
  2. Set the microscope shutter to use the camera:
    Camera shutter on Microscope 4 Nikon LS200, showing to pull slider OUT for camera imaging.

  3. Click the Camera in the software to begin acquiring images in the software:
    Click the link in the Camera List

Choosing the Calibration

In order for the software to know which objective you are using, you must choose the appropriately named Calibration from the dropdown lists. (eg. "5x, 10x, 20x etc."). Staff have set the software to display the Calibration/Objective name and microscope on the scale bar in the corner of the image.

There are 2 separate ways to choose the calibration to use. 

During live imaging

BEFORE snapping the image:

On the bar right above the image window, use the drop-down menu called Magnification, which is used for measuring live images. (This is also available from the top "Options" pull down menu). Choose the calibration from this list during live imaging, before clicking "Snap".

The Magnification dropdown menu.

Capture the Image

Then, when the live image is ready, click [Snap] to capture an image. This image retains internal knowledge of which calibration was used during the capture, and is loaded into a new Tab.

After Image Capture

To change the calibration applied on an image that has already been captured, use the top menu bar item:

Menu "Image" > "Resolution".  Choose the correct calibration setting for the objective lens you were using and you will now have calibrated measurements for the still image.

Menu item Image > Resolution popup window with scope calibrations dropdown.

Saving an Image with Measurements

Perform measurements as desired using the various measurement tools (scale bars, line lengths etc.).

If you want the measurements saved onto the still image, use the menu item:

Layers > Merge to Image... to merge the annotation layer onto the image. This will create a new image tab that has your measurements burned into it, the original editable tab is also still open.

Menu item Layers > Merge to Image... to burn the measurements into the image, before saving the image file.

The subsequent "Export options" window allows you to choose the Font size for the text etc. "50" is typical.

Save your pictures as jpegs as normal with File > Save As....

You can save all open tabs into a folder with automatic naming using File > Batch Save.

End Session

  1. Set the Camera shutter back to Eyepiece imaging (pushed IN) when done.
  2. Close all your image tabs in AmScope (Right-click the "Live" tab and choose "Close all but this"), or just close the software.
